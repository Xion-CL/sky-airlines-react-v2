# Sky Airlines - Sistema de Reservas

Proyecto académico para la asignatura de Programación Front End.

## Características

- ✅ CRUD completo de reservas
- ✅ Validación de RUT chileno
- ✅ Integración con API mindicador.cl
- ✅ Cálculo de precios en UF, USD, EUR y CLP
- ✅ Encriptación de datos sensibles
- ✅ Hashing de contraseñas con Salt
- ✅ Calculadora de edad
- ✅ Autenticación de usuarios
- ✅ Diseño responsive con Bootstrap
